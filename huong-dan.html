2. Hướng Dẫn Cài Đặt
01Tải và cài đặt XAMPP

Tải XAMPP từ trang chủ Apache Friends và cài đặt trên máy tính.

Hình 1: Trang tải XAMPP tại apachefriends.org

  1. 1
    Truy cập trang chủ XAMPP

    Vào https://www.apachefriends.org → Chọn phiên bản PHP 8.x → Nhấn Download

  2. 2
    Chạy file cài đặt

    Nhấp đúp file xampp-installer.exe → Chọn thư mục cài đặt (mặc định C:\xampp) → Nhấn Next đến khi hoàn tất.

  3. 3
    Khởi động Apache và MySQL

    Mở XAMPP Control Panel → Nhấn Start bên cạnh Apache và MySQL → Đảm bảo cả hai đều chuyển sang màu xanh.

Lưu ý cổng: Apache mặc định cổng 80, MySQL cổng 3306. Nếu bị xung đột, vào Config → httpd.conf để đổi cổng.
02Copy source code vào htdocs

Giải nén và đặt thư mục dự án vào đúng vị trí để Apache nhận diện.

Terminal / File Explorer
# Đường dẫn đích cần đặt dự án: C:\xampp\htdocs\carrental_test\ # Cấu trúc sau khi copy: C:\xampp\htdocs\ └── carrental_test\ ├── index.php ├── database.sql ├── assets\ ├── config\ ├── controllers\ ├── models\ ├── views\ └── admin\
Quan trọng: Tên thư mục phải là carrental_test (viết thường, không dấu cách). URL sẽ là http://localhost/carrental_test/

Hình 1: Trang tải XAMPP tại apachefriends.org

03Import cơ sở dữ liệu

Tạo database và import file SQL vào phpMyAdmin.

Mở phpMyAdmin

Truy cập http://localhost/phpmyadmin trên trình duyệt.

Tạo Database mới

Nhấn New (cột trái) → Đặt tên car_rental → Chọn Collation utf8mb4_unicode_ci → Nhấn Create.

Import file SQL

Chọn database car_rental → Tab Import → Nhấn Choose File → Chọn file database.sql trong thư mục dự án → Nhấn Go.

Kiểm tra kết quả

Sau khi import thành công, bạn sẽ thấy các bảng: cars, car_images, bookings, services, admins.

04Cấu hình kết nối database

Chỉnh sửa file cấu hình để kết nối với MySQL.

Mở file config/database.php và cập nhật các thông số:

PHP — config/database.php
define('DB_HOST', 'localhost'); // Địa chỉ MySQL define('DB_PORT', '3306'); // Cổng MySQL (mặc định 3306) define('DB_NAME', 'car_rental'); // Tên database define('DB_USER', 'root'); // Tên người dùng MySQL define('DB_PASSWORD', ''); // Mật khẩu (để trống nếu dùng XAMPP mặc định)
XAMPP mặc định MySQL username là root, password để trống. Nếu bạn đã đặt mật khẩu, điền vào trường DB_PASSWORD.

4.2 Xem Danh Sách Xe

Khách hàng có thể duyệt danh sách xe với đầy đủ thông tin: hình ảnh, thương hiệu, năm sản xuất, số ghế, loại hộp số và giá thuê mỗi ngày.

  • 1
    Truy cập trang xe

    Nhấn Xe Cho Thuê trên thanh menu điều hướng hoặc nhấn Xem tất cả xe từ trang chủ.

  • 2
    Lọc và tìm kiếm

    Sử dụng thanh tìm kiếm để lọc xe theo tên, thương hiệu hoặc loại xe.

  • 3
    Xem chi tiết xe

    Nhấn vào card xe để xem đầy đủ thông tin: ảnh chi tiết, tính năng, mô tả và giá thuê.

4.3 Đặt Xe

Quy trình đặt xe đơn giản, nhanh chóng chỉ với vài bước.

Chọn xe muốn thuê

Từ trang danh sách hoặc chi tiết xe, nhấn nút "Đặt xe ngay".

Điền thông tin đặt xe

Nhập họ tên, số điện thoại, email, ngày nhận xe, ngày trả xe và địa điểm nhận xe.

Xem tóm tắt & tổng tiền

Hệ thống tự động tính tổng tiền dựa trên số ngày thuê × giá thuê/ngày.

Xác nhận đặt xe

Nhấn "Xác nhận đặt xe". Hệ thống tạo mã đơn hàng và gửi thông tin xác nhận.

Lưu ý: Ghi lại Mã đơn hàng sau khi đặt xe thành công. Bạn sẽ cần mã này để theo dõi trạng thái đơn hàng.
4.4 Theo Dõi Đơn Hàng

Khách hàng có thể tra cứu trạng thái đơn hàng bất kỳ lúc nào bằng mã đơn.

  • 1
    Vào trang "Tra cứu đơn hàng"

    Nhấn Theo dõi đơn trên menu hoặc truy cập ?page=track_order.

  • 2
    Nhập mã đơn hàng

    Điền mã đơn đã nhận vào ô tra cứu → Nhấn Tra cứu.

  • 3
    Xem trạng thái đơn

    Hệ thống hiển thị: thông tin xe, ngày thuê, tổng tiền và trạng thái hiện tại (Chờ xác nhận / Đã xác nhận / Đang thuê / Đã trả xe).

7. Câu Hỏi Thường Gặp (FAQ)
Kiểm tra:
1. PHP đã bật extension pdo_mysql chưa (trong php.ini)
2. Database car_rental đã được tạo và import SQL chưa
3. Thông tin kết nối trong config/database.php có đúng không
4. Xem log lỗi tại C:\xampp\php\logs\php_error_log
Kiểm tra:
1. Thư mục assets/images/cars/ phải có quyền ghi (writable)
2. Trong php.ini kiểm tra upload_max_filesize và post_max_size (tăng lên 10M nếu cần)
3. Extension gd đã được bật chưa
Cổng 80 đang bị chiếm (thường do Skype hoặc IIS). Giải pháp:
1. Mở XAMPP Control Panel → Apache → Config → httpd.conf
2. Tìm dòng Listen 80 → đổi thành Listen 8080
3. Khởi động lại Apache → Truy cập http://localhost:8080/carrental_test/
